# Session 011 publication-safe evidence

This directory contains deterministic reports produced by the read-only
navigation-media analyzer:

- `navigation-media.public.json` - ISO/Joliet topology, validated FLDB
  structure and aggregate classifications;
- `firmware-media-contract.comparison.json` - conservative join with the
  Session 010 firmware-side optical contract and operational graph v4.

No map or firmware bytes, filenames, internal database names, raw metadata
strings, raw opaque fields, local paths or extracted resources are included.
The source ISO remains a local, ignored and provenance-unverified research
artifact.
