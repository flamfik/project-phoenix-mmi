This directory contains deterministic reports produced by the read-only
corrected parser-candidate analyzer:

- `cd1-fldb-candidate-dataflow.public.json` and
  `cd3-fldb-candidate-dataflow.public.json` - bounded block, register roles,
  relative pointer groups and counterevidence;
- `cd1-cd3.fldb-candidate-dataflow.comparison.json` - byte/hash and call-topology
  comparison;

strings, local paths, map payloads or extracted resources are included.
