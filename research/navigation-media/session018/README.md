# Session 018 public evidence

This directory contains deterministic, publication-safe reports from the
field-12 accessor call-family, zero-tail slot and registered optical-graph
intersection analysis.

Files:

- `cd1-accessor-dispatch.public.json`;
- `cd3-accessor-dispatch.public.json`;
- `cd1-cd3.accessor-dispatch.comparison.json`;
- `accessor-dispatch-correlation.json`.

The reports contain no firmware or instruction bytes, absolute runtime
addresses, raw strings, local paths, map payloads or extracted resources.

Result: 266 of 288 fixed CD3 accessor-call contexts converge on one CD1 target,
with 179 unique context matches. The CD1 target lies inside a confirmed run of
five `in-image pointer + 12 zero bytes` records. Runtime patch/trampoline
semantics remain a hypothesis. No data-only target occurrence, direct static
callback record or bilateral intersection with the 35 registered graph nodes
was found. Parser, sector ABI, buffer ownership/provenance and partition
consumer remain open.
