# Session 007 publication-safe evidence

This directory contains generated, publication-safe evidence for the cross-version reference graph and bounded owner assessment.

- `cd1-reference-graph.public.json` - MMI 5150 exact-anchor, descriptor and marker-profile summary;
- `cd3-reference-graph.public.json` - MMI 5570 exact-anchor, descriptor and marker-profile summary;
- `cd1-cd3.reference-graph.comparison.json` - normalized descriptor graph, structural marker comparison and owner-evidence decision.

The reports contain hashes, offsets, counts, fixed marker names and normalized relationships only. They do not contain firmware bytes, target-window bytes, resource bytes, raw HTML/URIs, arbitrary strings, raw runtime addresses or complete pointer-run value lists. Full local reports remain under the ignored `research/firmware-5570/work/session007/` directory.
