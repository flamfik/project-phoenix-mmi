# Session 009 publication-safe evidence

This directory contains generated evidence for the navigation and persistent-storage boundary pass.

- `cd1-navigation-storage-boundary.public.json` - MMI 5150 fixed-marker clusters, bounded SH-3 references and filesystem validators;
- `cd3-navigation-storage-boundary.public.json` - MMI 5570 equivalent evidence;
- `cd1-cd3.navigation-storage-boundary.comparison.json` - ordered cross-version marker bands and operational graph v2.

The reports contain only fixed marker identifiers, offsets, counts, structural validation results and confidence-graded conclusions. They contain no firmware bytes, arbitrary extracted strings, map payload, filesystem payload or raw runtime addresses. Full local reports remain under the ignored `research/firmware-5570/work/session009/` directory.
