# Session 005 publication-safe evidence

This directory contains generated, publication-safe evidence for the CD1/CD3 browser-resource bundle analysis.

- `cd1-resource-bundle.public.json` - MMI 5150 structural summary;
- `cd3-resource-bundle.public.json` - MMI 5570 structural summary;
- `cd1-cd3.resource-bundle.comparison.json` - cross-version equality and pointer-run deltas.

The reports contain hashes, offsets, counts and bounded hypothesis-test results only. They do not contain firmware bytes, resource bytes, raw HTML, raw URIs or arbitrary extracted strings. Full local reports remain under the ignored `research/firmware-5570/work/session005/` directory.
