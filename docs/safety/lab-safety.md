# Laboratory Safety Rules

- Never analyze the only copy of an artifact; verify hashes before and after handling.
- Mount images read-only or extract to disposable working directories.
- Do not execute unknown firmware binaries on the host system.
- Use an isolated virtual machine or emulator for runtime experiments.
- Do not connect experimental software to a vehicle network during early research.
- Do not alter bootloaders, calibration, Component Protection, immobilizer, or emergency-related systems.
- Before any future bench flashing, document power stability, recovery interface, known-good image, logs, and rollback procedure.
- Stop immediately when an artifact appears vehicle-specific or contains personal data.
